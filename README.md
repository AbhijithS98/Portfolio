A premium, modern, and fully responsive single-page personal portfolio website built to showcase my projects, skills, and experience as a Full Stack Developer.

## 🛠️ Technology Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## ✨ Key Features

- **Modern & Premium Design:** Sleek dark mode aesthetics with glassmorphism effects and tailored color palettes.
- **Fully Responsive:** Flawless experience across mobile, tablet, and desktop devices.
- **Smooth Animations:** Subtle micro-interactions and scroll animations powered by Framer Motion.
- **Data-Driven Architecture:** All portfolio content (projects, skills, experience) is driven by local static data files, making updates effortless.
- **SEO Optimized:** Semantic HTML and optimized structure for better search engine visibility.



## 📂 Project Structure

- `src/components/`: Reusable React components organized by `layout` and `sections`.
- `src/data/`: Static TypeScript data files containing all portfolio content.
- `public/`: Static assets like images and the resume PDF.
